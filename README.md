# Gene-Disease Association Database

## Project Description
This project documents the design of a relational database for gene-disease associations and the planned technical environment for implementation.

## Academic Objectives
- Learn how to document a technical project using GitHub
- Practice database schema design in PostgreSQL
- Organize sample gene and disease data
- Write SQL queries for association analysis
- Document the planned environment using VirtualBox, Linux, and PostgreSQL
- Show Python integration with PostgreSQL
- Introduce database version control concepts with Liquibase
- Prepare a short academic report about genes associated with a disease

## Project Status
Current status: documentation, initial database design, sample SQL data, SQL queries, environment notes, Python integration notes, and report draft completed.

## Repository Structure
- `README.md` — main project documentation
- `schema.sql` — database schema
- `data.sql` — sample data
- `queries.sql` — SQL queries for analysis
- `report.md` — academic report draft with references
- `environment.md` — technical environment documentation
- `requirements.txt` — Python dependencies for PostgreSQL work
- `python_pg_example.py` — example Python connection to PostgreSQL
- `liquibase.md` — notes about database version control with Liquibase

## Database Entities
- Genes
- Diseases
- Gene-Disease Associations
